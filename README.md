# Kaggle-ML-Portfolio
A centralized portfolio of machine learning solutions and exploratory data analysis (EDA) for Kaggle competitions. This repository documents my end-to-end workflow, including feature engineering, model optimization, and predictive analytics across various domains.

[1. IEEE-CIS Fraud Detection: Predictive Pipeline](./ieee-fraud-detection)

Developed an end-to-end machine learning solution to detect fraudulent financial transactions using a dataset of 500k+ records. This project focuses on handling extreme class imbalance and complex feature engineering to identify subtle patterns in user behavior.

Key Highlights:

Model: LightGBM with Stratified K-Fold Cross-Validation.
Feature Engineering: Created 20+ custom features including transaction frequency, time-deltas, and user-identity aggregations.
Performance: Achieved a 0.89 ROC-AUC in cross-validation.
Stack: Python, Pandas, LightGBM, Scikit-Learn.
